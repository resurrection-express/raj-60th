# Birthday Site Starter

Static scrapbook-style starter for **Happy Birthday Rajesh**.

## Files
- `index.html` — page structure
- `style.css` — scrapbook styling
- `script.js` — placeholder for later interactions

## Easiest free hosting
Use any of these:
- Netlify
- Vercel
- GitHub Pages

## Recommended media plan
- Main birthday video: upload to YouTube as **Unlisted**, then embed it
- Photos: add directly into the photo gallery later
- Extra videos: either embed individually or link out to a private folder/album

## What to replace later
- Hero text if you want a different vibe
- Video placeholder in `#video`
- Photo placeholders in `#photos`
- Extra video cards in `#extras`
- Final message in `#message`

## Nice next upgrades
- Click-to-expand photo lightbox
- Real image thumbnails
- Mobile polish after content is added
- Password gate if you want semi-private sharing
